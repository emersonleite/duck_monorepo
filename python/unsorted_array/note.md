# Vetores não ordenados:

Considera-se com relação ao vetor:

- Inserção na primeira posição livre;
- Pode ter tamanho fixo ou não;

## Operações

### Inserção

- Big-O - constante - O(1);

### Pesquisa

- Big-O - linear - O(n);

### Exclusão com remanejamento de elementos

- Big-O - O(2n) = O(n);

### Duplicatas

- Big-O - O(n);
